# Ohio Demo 19 — نسخه فارسی RTL

بازسازی دقیق [Ohio Demo 19: Classic Shop](https://ohio.clbthemes.com/demo19/homedemo19-elementor/) با HTML و CSS سفارشی — کاملاً فارسی و راست‌چین.

## ویژگی‌ها

- همان تصاویر، رنگ‌ها و چیدمان دمو اصلی Ohio
- فونت Vazirmatn برای فارسی
- طراحی واکنش‌گرا
- بدون نیاز به build

## اجرا

```bash
python3 -m http.server 8080
```

سپس `http://localhost:8080` را باز کنید.

## ساختار

```
index.html
assets/css/custom.css
assets/js/main.js
```
