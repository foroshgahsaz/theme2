# خانه دلنشین — قالب فروشگاه فارسی

نسخه فارسی و راست‌چین (RTL) الهام‌گرفته از [Ohio Demo 19: Classic Shop](https://ohio.clbthemes.com/demo19/homedemo19-elementor/)، ساخته‌شده با HTML و Tailwind CSS.

## ویژگی‌ها

- کاملاً فارسی و راست‌چین (`dir="rtl"` / `lang="fa"`)
- طراحی واکنش‌گرا (موبایل، تبلت، دسکتاپ)
- فونت **Vazirmatn** برای نمایش بهتر متن فارسی
- Tailwind CSS از CDN (بدون نیاز به build)
- بخش‌های کامل: هیرو، دسته‌بندی، محصولات، نمایشگاه، نظرات، مجله، خبرنامه و فوتر

## ساختار فایل‌ها

```
.
├── index.html              # صفحه اصلی
├── assets/
│   ├── css/
│   │   └── custom.css      # استایل‌های سفارشی (فونت، انیمیشن)
│   └── js/
│       └── main.js         # منوی موبایل، هدر چسبان، فرم خبرنامه
└── README.md
```

## اجرای محلی

فایل `index.html` را مستقیماً در مرورگر باز کنید، یا با یک سرور ساده:

```bash
# Python
python3 -m http.server 8080

# یا Node.js
npx serve .
```

سپس به آدرس `http://localhost:8080` بروید.

## سفارشی‌سازی

- **رنگ‌ها:** در بلوک `tailwind.config` داخل `index.html` قابل تغییر است
- **تصاویر:** از Unsplash استفاده شده؛ می‌توانید با تصاویر خود جایگزین کنید
- **محتوا:** تمام متن‌ها مستقیماً در `index.html` قابل ویرایش هستند
